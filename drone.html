CLIQUE PARA JOGAR

Mova o Mouse para Girar / Inclinar o Drone
W / S / A / D : Mover | Espaço / Shift : Subir e Descer

Controles do Drone

  • Mouse X : Rotacionar (Guinada)
  • Mouse Y : Inclinar para Frente / Trás
  • W / S : Impulso para Frente / Trás
  • A / D : Deslocamento Lateral (Strafing)
  • Espaço / Shift : Subir / Descer
